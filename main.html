Here is a simple currency converter using HTML, CSS and JavaScript. This converter only performs conversion between US Dollars, Euros and British Pounds. You can add more currencies using the same logic. Here's a sample implementation: HTML: ```html

Currency Converter

``` CSS (style.css): ```css body{ font-family: Arial, sans-serif; } .container{ margin-top:100px; text-align:center; } button{ margin-top:20px; } ``` JavaScript (converter.js): ```javascript var conversion_rates = {'USD_EUR': 0.84, 'USD_GBP': 0.76, 'EUR_USD': 1.19, 'EUR_GBP': 0.9, 'GBP_USD': 1.32, 'GBP_EUR': 1.11 }; function convertCurrency() { var inputVal = document.getElementById('inputVal').value; var initialCurrency = document.getElementById('initialCurrency').value; var finalCurrency = document.getElementById('finalCurrency').value; var outputVal = document.getElementById('outputVal'); var key = initialCurrency + '_' + finalCurrency; outputVal.value = (inputVal * conversion_rates[key]).toFixed(2); } ``` Here, we have hardcoded the exchange rates into a `conversion_rates` variable. In a real-world application, we would probably fetch this information from some API that provides real-time rates. When the "Convert" button is clicked, we find the appropriate conversion rate from the `conversion_rates` variable and multiply the input value by this rate to get the converted value.
